# FTTV
The FTTV Project is creating a faster implementation of the twitch.tv website.

## Contributing
Want to help out? [Check out our contribution guide.](https://github.com/fttv-project/fttv/blob/develop/CONTRIBUTING.md)

## Stack
* Build
	* Webpack 2
	* TypeScript
	* TSLint
* Frontend
	* React
	* Redux
	* SCSS
	* React Router
	* redux-observable
	* i18next
